## 📖 Overview
**MaxTec Group** is a modern, fully responsive corporate landing page built for a software development and technology agency. The project showcases the company's core services, pricing plans, portfolio of latest work, and a contact gateway, all wrapped in a sleek, dark-themed UI with vibrant orange accents.

> **Mission:** "Building the Future With Advanced Software."

## ✨ Key Features
* **Modern Dark UI/UX:** A visually striking dark theme that highlights content and reduces eye strain.
* **Service Showcase:** Detailed sections for Web & App Development, AI & Data Science, Cyber Security, UI/UX Design, Video Editing, and Maintenance & Support.
* **Interactive Pricing Plans:** Clear and engaging pricing cards (Standard, Professional, Custom).
* **Portfolio Gallery:** "Our Latest Work" section to display previous projects with categories.
* **Team & About Section:** Highlights company statistics and key management team members.
* **Internationalization (i18n):** Built-in support for multiple languages.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.

## 🛠️ Tech Stack
* **Framework:** [React.js](https://reactjs.org/)
* **UI Library:** [Material UI (MUI) v5](https://mui.com/)
* **Styling:** `@mui/material/styles`, `styled-components` (Custom CSS-in-JS)
* **Icons:** Material Icons
* **Translation:** `react-i18next`
